# Как изменить настройки сайта (WhatsApp, телефон, email, адрес)

После развёртывания на Vercel ваш сайт статический и не имеет базы данных.
Настройки хранятся в браузере пользователя (localStorage).

## Вариант 1: Через админку (для каждого браузера отдельно)

1. Откройте админку: https://aquachemistry.kg/admin
2. Войдите с паролем: `admin123`
3. Перейдите в "Настройки сайта"
4. Заполните:
   - WhatsApp номер (например: `+996555123456`)
   - Телефон (например: `+996 555 123 456`)
   - Email (например: `info@aquachemistry.kg`)
   - Адрес (например: `г. Бишкек, ул. Примерная 123`)
   - Часы работы (например: `Пн-Пт: 9:00-18:00`)
5. Нажмите "Сохранить"
6. Обновите главную страницу - настройки появятся

⚠️ **ВАЖНО**: Эти настройки сохраняются только в вашем браузере! Другие пользователи их не увидят.

## Вариант 2: Задать настройки по умолчанию в коде (для всех пользователей)

Отредактируйте файл `app/src/hooks/useSettings.ts`:

```typescript
export function useSettings() {
  const [settings, setSettings] = useState<Settings>({
    siteName: 'AQUAHIMIYA',
    whatsappNumber: '+996555123456',  // ← Ваш номер WhatsApp
    phone: '+996 555 123 456',         // ← Ваш телефон
    email: 'info@aquachemistry.kg',   // ← Ваш email
    address: 'г. Бишкек, ул. Примерная 123',  // ← Ваш адрес
    workingHours: 'Пн-Пт: 9:00-18:00', // ← Часы работы
  });
```

Затем:
```bash
cd app
git add src/hooks/useSettings.ts
git commit -m "Update default site settings"
git push
```

Vercel автоматически задеплоит изменения через 1-2 минуты.

## Вариант 3: Использовать PHP хостинг для настроек

Если у вас есть PHP хостинг, можете развернуть папку `php-site` туда:

1. Загрузите `php-site` на ваш PHP хостинг через FTP
2. Убедитесь что `php-site/api/data/` доступна для записи (chmod 755)
3. В файле `app/src/lib/api.ts` измените:
   ```typescript
   const BASE = 'https://вашдомен.kg/api';  // URL к PHP API
   ```
4. Закоммитьте и задеплойте

Тогда настройки будут храниться на сервере и будут общими для всех пользователей.

---

## Рекомендация

Для статического сайта на Vercel используйте **Вариант 2** - задайте настройки прямо в коде.
Это самый простой и надёжный способ.
